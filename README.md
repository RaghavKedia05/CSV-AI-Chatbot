# 📊 CSV AI Chatbot

Turn any CSV file into an AI-powered searchable chatbot using embeddings + FAISS.

## 🚀 Features
- Upload any CSV
- Semantic search using Sentence Transformers
- Fast vector search with FAISS
- Clean Streamlit UI

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- FAISS
- Sentence Transformers
